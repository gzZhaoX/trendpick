트렌드픽 아이폰 웹앱 사용법

1. 압축을 푼다.
2. 이 폴더를 아무 웹서버에 올린다.
   - 가장 쉬운 방법: Cloudflare Pages, Netlify, Vercel 같은 정적 호스팅 사용
3. 아이폰 Safari로 주소를 연다.
4. 공유 버튼 -> 홈 화면에 추가
5. 아이콘을 눌러 앱처럼 사용한다.

현재 버전 특징
- 샘플 데이터 기반
- 새로고침 가능
- 즐겨찾기 저장 가능
- 홈 화면 추가 가능
- 오프라인 캐시 기본 지원

다음 개선 후보
- 실제 뉴스 데이터 연결
- 실시간 키워드 API 연동
- 디자인 고급화
- 아이콘 변경
