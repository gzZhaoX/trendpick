트렌드픽 v2 사용법

이 버전에서 바뀐 점
- 실시간 뉴스 RSS를 이용해 키워드 자동 추출 시도
- 연결 실패 시 마지막 저장 데이터 또는 샘플 데이터로 자동 대체
- 검색, 즐겨찾기, 기사 링크, 공유 기능 추가
- 아이폰 화면에 더 맞게 디자인 개선

주의
- 실시간 데이터는 공개 RSS 변환 서비스(rss2json)를 사용합니다.
- 해당 서비스나 네트워크가 막히면 샘플/저장 데이터가 보일 수 있습니다.
- 즉, 완전한 실시간 전용 서버 버전은 아니고 "서버 없이 최대한 편하게 쓰는 버전"입니다.

GitHub Pages에 반영하는 방법
1. 기존 저장소의 파일을 이 zip 안 파일들로 덮어쓰기
2. GitHub 저장소에서 각 파일 수정 또는 다시 업로드
3. 잠시 기다린 뒤 페이지 새로고침
4. 아이폰 홈 화면 앱도 다시 열기
